# Part I: Development Environment

현대적인 개발 환경 구축에 필요한 핵심 기술들을 학습합니다.

## 학습 목표

**재현 가능성(Reproducibility)** 과 **결정론적 빌드(Deterministic Builds)** 를 통해 모든 개발자가 동일한 환경에서 작업할 수 있도록 합니다.

## Chapter 1: Introduction to Nix

함수형 패키지 관리자를 통한 완전 재현 가능한 개발 환경 구축

- **1.1 Programming Paradigm**: 명령형과 함수형 접근법 비교
- **1.2 Functional Programming**: 불변성, 순수함수, 참조투명성
- **1.3 Attribute Sets**: Nix 데이터 구조와 설정 관리
- **1.4 Nix Basics**: 기본 문법과 실습
- **1.5 Nix Core Concepts**: 패키지 관리 시스템 동작 원리
