# Introduction

본 playground 코스는 합성생물학 ⋅ 인공진화공학 연구실의 개발 환경 및 인프라 접근을 위한 기본기 학습을 위해 개설되었습니다.

학습 과정은 다음의 목차와 같이 진행되며, 각 챕터는 여러 브랜치에 분산되어 있어, 학습자 여러분들께서는 각자 이 저장소를 fork 하셔서 학습을 진행하시면 됩니다.

본 과정을 따라 가시면서, 학습자 여러분들께서는 최종적으로 하나의 documentation을 완성함으로써 이수가 완료됩니다.

## 학습 목표

- Nix 패키지 매니저와 생태계 이해
- Git을 활용한 버전 관리 및 협업
- 기본적인 암호화 개념 및 보안 실습
- 개발 환경 구성 및 관리

## 과정 구성

### Part I: Development Environment

각 챕터에서는 실습을 통해 개발 환경 구성에 필요한 핵심 도구들을 학습합니다.

### Appendices

운영체제별 특화된 도구들과 고급 설정 방법을 다룹니다.

**참고**: 각 챕터의 실습 과제는 해당 브랜치에서 확인할 수 있습니다.
